Raindrops Server & Admin v.1.12.23

RainDrops - программный комплекс для хранения данных. Чтобы начать им пользоваться, вам необходимо:

1. Скачайте приложение
2. С помощью Raindrops Admin создайте учетные записи
3. Проверьте конфигурацию
4. Передайте пользователям ключи и конфигурации
5. Запустите сервер.

На этой странице вы можете скачать сервер-приложение. После скачивания и настройки вы сможете полноценно пользоваться приложением. Не забудьте поставить файервол!
