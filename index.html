Raindrops EcoSystem 2023-2025

Raindrops Client

  • Настройка конфигурации
  • Авторезервирование
  • Создание и использование сертификатов для защиты файлов
  • TLS - сети
  • Машинная обработка данных
  • Обработка больших данных типа .csv, .xlsx, .db
  • Многоуровневое шифрование данных алгоритмами
  • WEB-клиент для совместной работы с файлами

Raindrops Server

  • Хранение паролей и сертификатов
  • Храниение защищенных файлов
  • TLS - сети
  • Нейросеть для обработки данных табличного и числового форматов
  • Двойное шифрование и контейнеризация данных
  • Визуально удобное логгирование событий и создание отчетов
  • Встроенная система обнаружения вторжений

Raindrops Admin

  • Добавление пользователей
  • Удаление пользователей
  • Конфигурирование системы
  • Точная настройка групповых политик
  • TLS - сети
  • Автоматическое шифрование и хеширование чувствительных данных

Raindrops Mail

  • Защита подключения через SSL
  • Настройка политик безопасности
  • Создание адресов и их использование
  • Логгирование и создание отчетности

Input&Output data

  • Шифрование файлов алгоритмом AES
  • Улучшенные алгоритмы поиска файлов на накопителях
  • Встроенный редактор файлов(аналог "nano" в Linux)
  • Сортировка по различным фильтрам и параметрам

Raindrops editor

  • Программирование собственных расширений для экосистемы
  • Редактор кода
  • Сборка и компиляция модулей
  • Просмотр ошибок в формате UTF-8

Raindrops Version Control

  • Контроль версий файлов
  • Конфигурация с облаком Raindrops
  • Git-подобный интерфейс
  • GUI приложение

Raindrops Proxy Connector

  • Автоматический поиск прокси по соответствию параметрам
  • Возможность добавить свои прокси сервера
  • Создание цепочек прокси
  • Настройка автовключения и выключения
